****************
ansible.snmp.set
****************

Notes
-----

.. note::
   - The SNMP set task will always return 'changed'



Examples
--------

.. code-block:: yaml

    - name: Set several individual OIDs
      ansible.snmp.set:
        oids:
        - oid: "SNMPv2-MIB::sysContact"
          iid: '0'
          value: "cidrblock @ {{ ts }}"
        - oid: "SNMPv2-MIB::sysLocation"
          iid: '0'
          value: "Office @ {{ ts }}"
      vars:
        ts: "{{ lookup('pipe', 'date -u +\"%Y-%m-%dT%H:%M:%SZ\"') }}"
